## TikTok App Review Notes

Use `Jarvis` as the app name everywhere: TikTok developer listing, public website, Terms, Privacy, and demo video. Do not use `Jarvis TikTok Policy Pages` or any other variant as the app name.

Only request scopes that are demonstrated in the video:

- `user.info.basic`
- `video.upload`
- `video.publish`

The demo video should show this website domain, the Terms and Privacy pages, the real TikTok sandbox authorization screen, the Jarvis posting UI, the final submit action, and the returned publish/status result.
