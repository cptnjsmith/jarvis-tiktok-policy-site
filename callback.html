This page is the registered redirect target for the Jarvis TikTok integration.

After TikTok redirects here, the authorization code and state will appear in the URL. Copy the full callback URL or the code value and return to Jarvis to complete token exchange.

What to do next

  1. Copy the full browser address from this page after TikTok redirects you here.
  2. Or copy just the code query parameter value.
  3. Use that code with the local Jarvis OAuth helper to exchange it for tokens.

Expected local command

python3 ai-prompts-small-business/scripts/tiktok_oauth.py exchange-code '<CODE_FROM_CALLBACK>' --write-env
